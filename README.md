# Vuash

Compartilhe uma mensagem acessível apenas uma vez.

## Instalação

1. Instalar gems necessárias:

- activerecord
- sinatra
- shotgun

2. Bundle

3. Start o server
$ be shotgun

4. Acessar o link a primeira vez
http://localhost:9393

5. Comentar a linha: load 'schema.rb' no arquivo `app.rb` # TODO fix it

6. Enjoy :)
